# Core ML Framework

- 머신러닝 모델을 앱에 적용시키기 위해 사용하는 프레임워크.

- 모든 모델에 대해 균일한 표현을 제공한다.

- 앱은 사용자 디바이스에서 Core ML APIs와 user data를 통해 예측을 하고, 보다 정교한 모델을 만들기 위해 트레이닝을 시킬 수 있다.

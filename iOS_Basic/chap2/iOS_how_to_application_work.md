# 앱은 어떻게 동작하는가?

- 앱은 여러개의 오브젝트로 이루어져있다.

- 오브젝트는 iOS에서 제공하거나, 필요에 따라 만든다

- 오브젝트는 서로 메시지를 보내면서 커뮤니케이션한다.

- 사용자가 버튼 클릭시 (상호작용) -> 버튼은 뷰 컨트롤러에게 터치 이벤트가 작동했음을 알린다. 

- iOS는 event Driven으로 작동한다. 
  - 이벤트에 의해 프로세스가 동작한다.
